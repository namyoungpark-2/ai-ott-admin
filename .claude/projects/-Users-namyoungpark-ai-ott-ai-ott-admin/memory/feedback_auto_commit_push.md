---
name: auto-commit-and-push
description: Always git commit + git push after completing file work
type: feedback
---

파일 작업이 완료되면 항상 git commit + git push까지 수행한다.

**Why:** 사용자가 작업 완료 후 매번 수동으로 commit/push하는 것을 원하지 않음. 작업 흐름을 자동화하고 싶어함.

**How to apply:** 파일 생성/수정/삭제 작업이 끝나면 변경사항을 git commit하고 원격에 push까지 완료한다.
